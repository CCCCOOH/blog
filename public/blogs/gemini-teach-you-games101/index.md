学习 GAMES101 这种硬核图形学课程，“效率”的关键不在于你记了多少字，而在于你如何将**数学推导**转化为**空间想象**，再转化为**代码实现**。

既然你手头有 PDF 和视频，我建议采取一种“**以作业为导向，以 PDF 为骨架**”的策略。

## 1. 记笔记的意义：你到底在记什么？

在图形学中，**死记硬背概念（如：什么是光栅化）是没有意义的**。笔记的真正作用只有两个：

* **记录“顿悟时刻”：** 视频里老师讲某个公式，你一开始不通，突然在一个点想通了。把那个点（例如你刚才问的  的来源）记在 PDF 旁边。
* **梳理数学逻辑：** 图形学很多变换（如 MVP 变换、透视投影）是环环相扣的。笔记是帮你建立逻辑链条，防止下次看作业代码时忘了“这行公式为什么这么写”。

### ❌ 错误的记笔记方式：

* 抄录 PDF 里的公式（PDF 里已经有了，浪费时间）。
* 截屏铺满文档（这叫收藏，不叫笔记）。

### ✅ 推荐的记笔记方式（在 PDF 上做减法）：

* **PDF 注释法：** 直接在 PDF 的对应页面做标注。比如在投影变换那页写上：*“这里  是因为齐次坐标归一化时除以了 ”*。
* **逻辑流程图：** 如果某节课推导很长，在纸上画出逻辑链路：*“观察空间 → 挤压视锥体 → 正交投影 → 屏幕空间”*。

## 2. 最高效率的学习路径：三部曲

### 第一步：倍速过视频，主抓“物理意义”

* **操作：** 1.5x - 2.0x 倍速看视频。
* **重点：** 不要纠结于每一个代数计算的加减乘除，而要听懂**“为什么要这么做”**。比如：为什么要引入齐次坐标？（为了平移和旋转能统一用矩阵乘法）。
* **标记：** 听不懂或觉得极其重要的地方，在 PDF 上标个星号，等下回头重看。

### 第二步：强制性“作业驱动”（核心效率来源）

* **真相：** 闫老师的课，**精华一半在视频，另一半在 Homework 里面**。
* **操作：** 每看完一个模块（比如变换、光栅化、着色），**立刻**去写对应的作业。
* **笔记时机：** 当你在写代码发现公式套不上去时，再翻 PDF 或视频。这时候你记下的笔记，才是你真正掌握了的知识。

### 第三步：建立“视觉直觉”

* 图形学是视觉的。当你推导出一个矩阵时，脑子里要有个 3D 空间。
* 如果想不通，试着用手比划一下：手掌是平面，手指是  轴。这种“肉体记忆”比笔记管用得多。

## 3. 给你的一份实操建议

| 资源 | 处理方式 |
| --- | --- |
| **视频** | 用于“理解”。听懂逻辑即可，不用反复看，除非要手撕作业。 |
| **PDF** | 它是你的**地图**。所有的补充说明、坑点、作业里的发现，都记在上面。 |
| **作业代码** | 它是你的**地基**。如果你能写出代码实现渲染器，你甚至不需要笔记。 |

**总结建议：**
不要为了记笔记而记笔记。如果你觉得看 PDF 就能回想起老师讲的内容，那就**不记**；如果你遇到像  这种“第一眼看过去会懵”的点，就**在 PDF 对应位置写一句话总结**。
